About the Chatbot

This chatbot is a simple rule-based web application designed to provide information about traditional Nepalese foods and their ingredients. It helps users quickly learn what popular Nepali dishes are made of through an easy and interactive chat interface.

The chatbot was created as a learning project to understand how chat systems work using JavaScript and Node.js. It demonstrates concepts such as user input handling, message matching, DOM manipulation, and basic conversational logic.

By focusing on Nepalese cuisine, the chatbot also promotes local food culture and makes it easier for users to explore traditional dishes like momo, dal bhat, sel roti, and thukpa.

This project showcases basic frontend and backend integration using Bootstrap, custom CSS, and JavaScript.
